Writing Prompts for AI Agents

The more specific you are, the better the result!

Vague

"Create a tool to help with meetings"

AI doesn't know what kind of tool, for whom, or what features

Better

"Create a meeting action tracker where users can add action items with description, owner, and due date. Include a way to mark items complete."

Has features, but missing context and design details

Specific

Create a meeting action tracker for our UX team.

Features:
• Add items: description, owner, due date
• Checkbox to mark complete
• Filter: all / pending / done
• Data persists in localStorage

Design: Clean, minimal, IKEA blue (#0058A3), mobile-friendly, dark mode toggle.

Context, features, technical needs, design specs!

Formula: WHO is it for + WHAT it does + HOW it should work + WHAT it looks like

VS Code: Windows & Folders

Understanding how VS Code organizes your work

Windows

File → New Window (⌘⇧N)

Opens a fresh VS Code instance

Multiple windows = Multiple projects

Each window has its own folder

Drag tabs between windows

Compare files side by side

Folders = Projects

File → Open Folder (⌘O)

Select your project folder

One folder = One project

All files inside are visible

Create new folder → Start fresh!

Right-click in Finder/Explorer

How Copilot Sees Your Folder

When you open a folder, Copilot can "see" all files inside. It understands your project structure, reads your code, and suggests changes that fit your existing patterns. The folder IS the context.

Understanding File Types

What are all these files? A quick guide for designers

.html

Structure
The skeleton of a page. Headings, paragraphs, buttons, links.

.css

Styling
Colors, fonts, spacing, layout. The visual design!

.js

Behavior
Interactivity! Clicks, animations, data handling.

.json

Data/Config
Settings, data files, package info.

.tsx / .jsx

React components (HTML + JS together)

.md

Markdown (documentation, README)

.svg

Vector graphics (icons, logos)

Don't worry about memorizing! Copilot knows what each file type needs.

What is CLI?

CLI = Command Line Interface
Text commands instead of clicking buttons.

cd = Change directory (folder)

npm = Package manager

npm run = Start something

Ctrl+` = Open terminal in VS Code

Good news: Copilot can run terminal commands for you!
